## **Setup Instructions**

1. Clone the repository:
   ```bash
   git clone https://github.com/thanh-pham2k/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the project:
   ```bash
   ng serve
   ```
